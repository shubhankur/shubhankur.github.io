Salozone Home Salon Services

Made: April, 2019

Developed a website (salozone.com) through which we received bookings from users around the city and then allotted the bookings to the corresponding service person. Used HTML, CSS, JS and Bootstrap for UI and later implemented React JS to make it fast and scalable. Back-end is developed in NodeJs and its framework ExpressJs. Database is in MongoDB. Front-end code is published in Surge and all the images in the website is fetched through Cloudinary, to decrease the cost of the database and the loading time of the website. Service is deployed in the Heroku. Slot booking and allotment of beautician is handled through a customized scheduling system where it searchesthrough the database and list out the available beauticians according to the services booked and the chosen time, after which it maps the booking to one of the beautician based on parameters like location, type of services etc. It uses pattern recognition and data analytics to study user purchase behaviour. It also uses an Elo-rating system which prioritizes customers and beautician according to the ratings and rolls out customized offers and pricing. Incorporated a recommendation system using collaborative filtering in order to provide users' most preferred and suitable services on top of the homepage. We employed more than 30 women and Salozone.com is the most visited website (in its category) in the city, generating a revenue of more than INR 70 lacs in three years.

Smart Sorting System

Made: April, 2019

A multi-level classification system designed for sorting fruits of different types and grades depending on their chemical and visual composition. It uses a CNN based classification model to segregate different fruits visually. Then, it uses a weight sensor, ethylene gas sensor to separate the fruits into different grades. The goods to be tested are placed on a conveyor belt, the weight sensor activates the device based on the weight. To grade the fruits we use a classification model which takes the ethylene gas level and fruit weight as input and grades the fruit into different types.

Room Occupancy Counter

Made: March, 2017

Developed a device to keep track of the number of people in a room using Arduino and Ultrasonic sensors for the subject Sensors and Instrumentation. The main purpose of the device was to save electricity and prevent overcrowdedness. It counts the number of people inside the room and notifies the admin with a buzzer if some extra electric appliance is still open or if number of people in the room exceeds the maximum limit and also if the room getsempty. Lead a team of 4 people and presented the project in a seminar for which we received appreciation and were awarded an ‘A’ grade.
